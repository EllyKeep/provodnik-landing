# 2. ПОЛНЫЙ КАТАЛОГ УСЛУГ

## 2.1 Чат-боты и мессенджер-автоматизация

### WhatsApp-бот
**Что это:** AI-ассистент в WhatsApp, который отвечает клиентам, принимает заявки, квалифицирует запросы и работает круглосуточно.
**Кому подходит:** любому бизнесу с входящим потоком клиентов через WhatsApp.
**Задачи:** первичный контакт, сбор данных, ответы на FAQ, передача горячих лидов.
**Польза:** менеджер не тратит время на рутинные ответы, клиент получает ответ мгновенно.
**Когда рекомендуется:** если WhatsApp — основной канал связи с клиентами.

### Telegram-бот
**Что это:** автоматизированный ассистент в Telegram с кнопками, меню, регистрацией, уведомлениями.
**Кому подходит:** онлайн-школам, конференциям, HR, сервисным компаниям.
**Задачи:** регистрация, рассылки, FAQ, квалификация, выдача документов и билетов.
**Польза:** полная автоматизация входящих без участия сотрудника.
**Когда рекомендуется:** если аудитория активна в Telegram или нужен сложный сценарий с кнопками.

### Бот-ассистент для регистрации (мероприятия, конференции)
**Что это:** бот, который полностью ведёт регистрацию участников: собирает данные, подтверждает участие, выдаёт электронный билет, отвечает на вопросы по программе.
**Кому подходит:** организаторам конференций, форумов, онлайн-мероприятий.
**Задачи:** замена ручной регистрации, снижение нагрузки на организатора.
**Польза:** ноль потерянных регистраций, мгновенное подтверждение, работа 24/7.

### Бот для бронирования (рестораны, клиники, салоны)
**Что это:** автоматизированная система приёма брони через мессенджер с подтверждением и напоминаниями.
**Кому подходит:** ресторанам, клиникам, салонам красоты, стоматологиям.
**Задачи:** принять бронь, подтвердить, напомнить за нужный период, снизить no-show.
**Польза:** менеджер свободен от телефона, клиент получает мгновенное подтверждение.

### Бот для онлайн-записи
**Что это:** интегрированная система записи через бот с выбором времени и специалиста.
**Кому подходит:** медицинским учреждениям, образовательным центрам, мастерам.
**Польза:** доступна запись 24/7, снижение нагрузки на администратора.

## 2.2 Голосовые AI-агенты

### Voice AI Agent (VIP)
**Что это:** голосовой агент на основе Vapi.ai + ElevenLabs, который отвечает на звонки на итальянском языке 24/7, как живой оператор.
**Кому подходит:** бизнесу с высоким чеком: клиники, медцентры, premium-сервисы.
**Задачи:** ответить на вопросы о ценах, протоколах, записи, снять нагрузку с врачей/операторов.
**Польза:** клиент получает качественный ответ в любое время суток, без очереди.
**Когда рекомендуется:** если входящих звонков много, а команда не справляется или работает не круглосуточно.

## 2.3 AI-агенты и ИИ-ассистенты

### AI-агент для продаж
**Что это:** интеллектуальный агент, который квалифицирует входящих лидов по заданным критериям и передаёт менеджеру только «горячих» клиентов.
**Кому подходит:** автосалонам, застройщикам, B2B-компаниям с высоким чеком.
**Задачи:** задать 5–7 квалифицирующих вопросов, оценить намерение, передать данные в CRM.
**Польза:** менеджер работает только с реальными покупателями.

### AI-скрининг кандидатов (HR)
**Что это:** бот, который автоматически проводит первичное собеседование: собирает анкету, задаёт вопросы, оценивает соответствие критериям и передаёт HR только подходящих кандидатов.
**Кому подходит:** HR-агентствам, компаниям с активным наймом.
**Задачи:** снизить нагрузку на HR при большом потоке анкет (200–500/неделю).
**Польза:** экономия до 80% времени на первичный отбор.

### AI-рекрутер
**Что это:** агент, который ведёт переписку с кандидатами, отвечает на вопросы о вакансии, согласовывает интервью.
**Кому подходит:** кадровым агентствам, HR-отделам крупных компаний.

### AI-менеджер по клиентскому сервису
**Что это:** агент, который обрабатывает входящие вопросы клиентов, решает типовые задачи, при нестандартных — передаёт оператору.
**Польза:** снижение нагрузки на call-центр / поддержку.

### AI-консультант
**Что это:** бот на базе базы знаний компании, который отвечает на вопросы клиентов о продуктах, услугах, ценах, условиях — как опытный менеджер.

### AI-операционист
**Что это:** агент, автоматизирующий внутренние бизнес-процессы: сбор данных, заполнение форм, маршрутизация задач.

## 2.4 Автоматизация бизнес-процессов

### Автоматизация заявок
**Что это:** настройка цепочки: заявка → бот → CRM → уведомление менеджеру → запись в таблицу.
**Польза:** ни одна заявка не теряется, всё фиксируется автоматически.

### Автоматизация CRM
**Что это:** интеграция бота с CRM (Bitrix24, AmoCRM, Notion, Google Sheets и другими) — автоматическое создание и обновление карточек.

### Интеграции
**Что это:** подключение мессенджеров, форм, CRM, email, Google Sheets, Zapier, Make между собой.
**Кому подходит:** компаниям, у которых данные разбросаны по разным системам.

### Автоматические напоминания и уведомления
**Что это:** цепочки сообщений: напоминание о встрече, о вебинаре, о записи, об оплате.
**Польза:** снижение no-show, рост конверсии без дополнительного персонала.

### Автоматизация лид-воронки
**Что это:** настройка автоматической цепочки от первого контакта до передачи менеджеру: бот → квалификация → прогрев → передача.

# 5. ЦЕЛЕВАЯ АУДИТОРИЯ

## 5.1 Малый и средний бизнес (Италия, Европа)

**Кто это:** предприниматели с командой от 2 до 50 человек, у которых есть поток клиентских обращений, но нет ресурсов нанимать дополнительных сотрудников только для обработки входящих.

**Боли:**
- Менеджер перегружен типовыми вопросами и упускает важных клиентов
- В нерабочее время никто не отвечает — клиент уходит к конкуренту
- Заявки теряются или обрабатываются слишком медленно
- Сложно масштабироваться без роста штата

**Задачи:**
- Автоматизировать первичный контакт
- Не терять входящие заявки
- Освободить менеджера для сложных задач

**Решения Provodnik:** WhatsApp-бот / Telegram-бот / AI-агент для квалификации лидов / автоматизация CRM

---

## 5.2 Медицинские учреждения (клиники, стоматологии, центры)

**Кто это:** частные клиники, стоматологии, эстетические центры, центры репродуктивной медицины.

**Боли:**
- Регистратор 80% времени на телефоне
- Пациенты в стрессе задают однотипные вопросы в любое время
- Запись через звонок неудобна, часть пациентов уходит не дозвонившись
- Список ожидания не управляется системно

**Задачи:**
- Автоматизировать запись и напоминания
- Снизить нагрузку на администратора

**Решения Provodnik:** бот для записи, бот для листа ожидания, Voice AI Agent (VIP для PMA и premium-клиник)

**Решения Provodnik:** AI-скрининг кандидатов, бот-рекрутер
